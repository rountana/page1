Travel booking app using Python/ Flask/ Fast API

Build a travel website which will offer the following features

1. Allow user to search for hotels based on date, destination and number of travellers

Take the results of the API search and display in a organized format.
- Hotel Name
- Hotel pictures
- Daily Price
- Total Price

2. Fetch detailed hotel information if user clicks on a particular hotel
- Hotel room information
- facilities in each room

3. Book room and finalize payment (no actual payment processing is required, just display message payment successful when clicked on pay)

Use the Amadeus Self Service API

Refer the amadeus API documentation below :
https://developers.amadeus.com/self-service/category/hotels/api-doc/hotel-list

Lets build with Python/ Flask and Fast API.
